2026软件项目开发比武考核标准
基本信息
- 开发要求：自选框架、编程语言、开发工具；支持智能体开发（Trae、CodeBuddy、Claude Code等）
- 系统选题范围（不限类型）：生产运营类、客户服务类、物流仓储类、金融服务类、海洋运输类、医养健康类、企业管理类、协同办公类、数据智能类等
- 总分构成：系统设计20分 + 项目开发75分 + 项目部署5分 = 100分
一、系统设计（总分：20分）
考核要求
结合自选业务方向完成系统设计，实现完整业务闭环、数据正常流转、业务逻辑自洽；需满足指定码表、功能菜单数量要求，内容与选题无关不得分。

二、项目开发（总分：75分）
2.1 基础功能（10分）
功能描述
实现系统基础通用操作及表格高级功能，覆盖数据基础管理与表格自定义、导出能力。
得分细则
- 基础数据操作（4分）：查询、新增、修改、删除，每完成1个得1分，全部完成满分
- 表格高级功能（6分）：表格列排序、表头自定义显示、Excel导出，每完成1个得2分，全部完成满分
2.2 表单数据导入功能（10分）
功能描述
支持Excel表单数据批量导入，具备数据格式精准校验、错误位置页面提示、重复数据自动去重能力，保证入库数据唯一有效。
得分细则
- 数据校验与错误提示（4分）：实现Excel批量导入、自动校验数据格式、页面精准提示错误位置（例：第10行第1列身份证位数缺失）
- 重复数据处理（6分）：自动识别Excel重复数据，导入过程自动去重，数据库无重复记录
2.3 监控与审计功能（10分）
功能描述
搭建全量日志管理与审计体系，自动记录用户关键操作、接口请求、系统异常；支持多条件日志筛选、链路追踪、系统与日志数据可视化展示。
得分细则
- 日志收集与多条件查询（2分）：日志包含入参、出参、请求地址、操作人、操作时间；支持多条件组合查询、分页展示
- 日志链路追踪（2分）：关联业务功能，日志详情可查看对应接口执行的全部SQL语句
- 数据可视化展示（6分）
  - 4分：大屏图表展示系统状态（CPU、内存等）、日志数据（操作频次、异常分布等）
  - 2分：大屏包含至少6个指标项，支持数据实时更新
2.4 消息中心功能（5分）
功能描述
搭建完整消息中心，支持消息模板配置、模板引用发送、首页消息提醒、消息撤回与重发全流程能力。
得分细则
- 1分：消息管理页面支持自定义配置消息模板
- 1分：消息发送页面可引用已有模板，完成消息发送
- 1分：系统首页展示消息提醒
- 2分：支持消息撤回、消息重发功能
2.5 海量表格数据加载功能（10分）
功能描述
支持10万条以上海量数据批量导入、无分页滚动加载，保证页面流畅不卡顿、无空白、无浏览器卡死，展示接口响应耗时。
得分细则
- 4分：实现异步导入10万条以上数据，导入成功且数据完整
- 4分：列表无分页展示海量数据，下拉滚动加载，页面无卡顿、无空白、浏览器无响应异常
- 2分：页面实时展示接口请求响应时间
2.6 用户画像功能（5分）
功能描述
基于业务大数据构建用户画像，通过指定图表可视化展示数据，支持业务数据变更后画像数据实时联动更新。
得分细则
满分5分：需实现标签云、环形图、雷达图三类图表展示用户画像；调整业务数据后，画像图表数据自动联动更新，全部达标得分。
2.7 工作流功能（10分）
功能描述
基于业务场景搭建工作流，实现多节点流转、全维度审批操作、BPMN流程图可视化展示。
得分细则
- 4分：配置并展示4个审批节点，可完整演示全流程审批流转
- 4分：支持节点审批、驳回、会签、指定回退、转办、终止全部操作功能
- 2分：通过BPMN标准业务流程图可视化展示所有流程节点与流转关系
2.8 智能表单填写插件功能（5分）
功能描述
自主开发原生浏览器插件（禁止油猴等现有插件），具备多字段校验、表单模板保存、一键自动填充能力。
得分细则
- 3分：页面新增校验按钮，点击可校验至少5个表单字段，精准展示校验结果与异常提示
- 2分：校验通过后可将表单数据保存为自定义模板，新表单页面可选择模板一键填充
2.9 工作台小组件功能（5分）
功能描述
搭建自定义首页工作台，支持拖拽布局、小组件管理，具备基础交互业务能力。
得分细则
- 3分：支持工作台拖拽布局、布局初始化、小组件添加、布局保存功能
- 1分：工作台自定义小组件数量不少于5个
- 1分：至少1个小组件具备业务交互能力（查询、审批、消息提醒任选其一）

2.10 API服务市场功能（5分）
功能描述
搭建API服务管理市场，支持API全生命周期管理及在线调试、上下架权限控制。
得分细则
- 2分：支持新增发布API（名称、描述、接口地址等），发布后可在线调试、验证接口可用性
- 1分：支持API上下架操作，下架后接口不可调用
- 2分：支持已发布API的编辑、删除操作
三、项目部署（总分：5分）
考核要求
参赛选手独立完成项目服务器部署，部署后系统可正常访问，所有核心功能运行正常、无异常报错。
评分细则
部署成功、系统可正常运行、所有功能均可正常使用，得满分5分；功能异常、部署失败酌情扣分。
