# CRM 智能表单助手 — 浏览器插件

对应比武项目 **2.8 智能表单填写插件功能**（5分），原生 Chrome/Edge 扩展，禁止油猴。

## 得分覆盖

| 得分点 | 分值 | 实现 |
|--------|:----:|------|
| 页面新增校验按钮，校验 ≥5 个表单字段，精准展示结果与异常提示 | 3分 | ✅ 浮动工具栏「校验表单」按钮 → 扫描 dialog 中所有字段 → 13 条校验规则 → 浮层展示逐字段通过/失败/跳过 + 汇总卡片 |
| 校验通过后保存为自定义模板 + 新表单可选择模板一键填充 | 2分 | ✅ 全部通过后直接保存 / 单独「保存模板」/「选择模板」→ 一键填充所有字段 |

## 安装方法

### Chrome / Edge / 360 极速浏览器

1. 打开浏览器扩展页面：
   - Chrome：地址栏输入 `chrome://extensions`
   - Edge：地址栏输入 `edge://extensions`
2. 打开 **「开发者模式」**（右上角开关）
3. 点击 **「加载已解压的扩展」**
4. 选择本文件夹 `browser-plugin/`
5. 打开 CRM 系统 → 进入任意表单页面（客户/商机/合同）

## 使用说明

### 1️⃣ 校验表单
- 点击 CRM 页面的 **新增/编辑** 按钮，打开表单弹窗
- 页面右上角出现浮动工具栏「表单助手」
- 点击 **✅ 校验表单**
- 弹窗显示逐字段校验结果（通过/失败/跳过），含错误原因
- 全部通过时可一键「保存为模板」

### 2️⃣ 保存模板
- 方式一：校验全部通过后，点击结果浮层底部的「保存为模板」
- 方式二：直接点击工具栏的 **💾 保存模板**，输入名称即可

### 3️⃣ 提交拦截
- 点击弹窗底部 **确定/保存**（主按钮）时自动执行与「校验表单」相同的规则
- 存在 **失败** 项时阻止提交，并弹出校验结果

### 4️⃣ 一键填充
- 在新表单页面，点击工具栏 **📂 选择模板**
- 点击目标模板的 **「一键填充」** 按钮
- 所有匹配字段自动填入

## 校验规则（13 条规则，覆盖全部 CRM 表单）

| 字段 | 必填 | 规则 |
|------|:----:|------|
| 客户名称 | ✅ | 2~100 字符 |
| 商机名称 | ✅ | 2~100 字符 |
| 合同名称 | ✅ | 2~100 字符 |
| 联系电话 | ✅ | 11 位数字（如 12345678910） |
| 邮箱 | ❌ | 标准邮箱格式（选填） |
| 所属行业 | ✅ | 不能为空 |
| 客户等级 | ✅ | 不能为空 |
| 商机阶段 | ✅ | 不能为空 |
| 省份 | ✅ | 不能为空 |
| 预计金额 | ✅ | 须大于 0 |
| 合同金额 | ✅ | 须大于 0 |
| 详细地址 | ❌ | 选填，建议 ≥5 字符 |
| 合同编号 | ✅ | 不能为空 |
